# Dynamic-Hedge-Ratio-Pairs-Trading-using-Kalman-Filter-German-Equity-Market
State-Space Modeling for Adaptive Statistical Arbitrage in European Markets

Overview

This project implements a state-space based statistical arbitrage strategy for the German equity market.

Unlike traditional pairs trading, this model estimates time-varying hedge ratios using a Kalman Filter, allowing the strategy to adapt to evolving market relationships.

Features

• Dynamic hedge ratio estimation
• State-space modeling
• Cointegration testing
• Z-score mean reversion strategy
• Full backtest pipeline
• Professional visualization
